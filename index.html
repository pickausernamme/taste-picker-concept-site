Hexadecimal color pickers work like this - there are 3 colors (red, green and blue) and each one of them has an assigned hexadecimal number going from 00 to FF (the colors can go from 0 to 255).

 For example FFFFFF is 255, 255, 255 (white) and 000000 is 0, 0, 0 (black) in RGB.



We could also assign hexadecimal values. They could go in this order - Sweet, Salty, Sour, Umami, Bitter.
